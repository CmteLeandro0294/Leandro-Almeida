# Matemática do Lorenzo

Jogo educativo de matemática com sistema de múltiplos temas para crianças.

## Funcionalidades

- **3 Temas Personalizados**: Mago, Piloto ou Cientista
- **4 Operações Matemáticas**: Soma, Subtração, Multiplicação e Divisão
- **Dicas Visuais Animadas**: Representação visual de cada operação
- **Sistema de Pontuação**: 10 pontos por acerto, 5 com dica
- **Progressão de Níveis**: 25 níveis com dificuldade crescente

## Como Executar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## Tecnologias

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
