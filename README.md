# Leandro-Almeida

Exemplos de páginas HTML para cotação de voo executivo.

- **cotacao_v2.html** – interface simples com atualização automática de tarifa, conversão entre NM/KM e módulo básico de comissão.
- **cotacao_v3.html** – versão avançada com tema escuro, filtros para geração de PDF, campos extras e cálculo de comissão opcional.

Abra qualquer arquivo no navegador para utilizar.
